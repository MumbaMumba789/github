# 📊 Data Jobs Insights Dashboard

An interactive Power BI dashboard that analyzes data job postings to uncover trends in salaries, job demand, required skills, and hiring companies.

---

## 📌 Project Overview

This project explores the current data job market using Power BI. The dashboard enables users to identify high-paying roles, the most in-demand technical skills, and salary differences based on education requirements.

The report is designed to help aspiring data professionals, recruiters, and businesses better understand the data job landscape through interactive visualizations.

---

## 🎯 Objectives

- Analyze average salaries across data-related roles.
- Identify the most in-demand technical skills.
- Compare salaries for jobs requiring a degree versus those that do not.
- Discover companies offering the highest salaries.
- Explore relationships between job demand and salary.
- Build an interactive dashboard with drill-through functionality.

---

## 🛠 Tools & Technologies

- **Power BI Desktop**
- **Power Query**
- **DAX**


---

## 📊 Dashboard Features

### Main Dashboard

- Key Performance Indicators (KPIs)
  - Average Hourly Salary
  - Average Yearly Salary
  - Total Companies
  - Total Job Postings
  - Total Skills

- Highest Paying Companies

- Average Salary by Job Title

- Interactive navigation to detailed skill analysis

---

### Skills Drill-Through Dashboard

Provides detailed analysis for individual technical skills, including:

- Average salary
- Job postings
- Median hourly salary
- Degree vs No Degree salary comparison
- Job postings vs salary scatter plot
- Detailed skills table

---

## 📈 Key Insights

Some insights discovered from the analysis include:

- SQL and Python are the two most requested technical skills.
- Spark and AWS command some of the highest salaries.
- Senior Data Scientists and Senior Data Engineers earn the highest average salaries.
- Degree requirements do not always lead to significantly higher salaries for every skill.
- Certain niche technologies offer high salaries despite relatively lower demand.
